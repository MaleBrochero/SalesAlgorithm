Puntos de Ventas – RiwiTechStore

Tecnologias usada:
Python

Este proyecto es un programa básico en Python que permite registrar una venta desde consola.
Fue desarrollado como una primera versión para practicar lógica de programación, uso de variables, condicionales y manejo básico de errores.
El sistema solo permite registrar una venta por ejecución.

📌 ¿Qué hace el programa?

El programa:

Solicita el nombre del cliente
Solicita el nombre del producto
Pide el valor del producto
Pide la cantidad de unidades
Pregunta si el cliente es VIP
Calcula el total
Aplica descuento si corresponde
Muestra el resultado final en pantalla

🧮 Regla de descuento

Si el cliente es VIP (opción 1), se aplica un 10% de descuento sobre el total de la compra.
Si no es VIP (opción 2), paga el valor completo.

▶️ Cómo ejecutar el programa

Debes tener instalado Python 3.

En la terminal escribe:

python nombre_del_archivo.py
Luego solo sigue las instrucciones que aparecen en pantalla.
